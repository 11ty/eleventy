## Installation

```
npm install @11ty/client --save
```

With exports for:

- `@11ty/client`
- `@11ty/client/md` (Markdown Template Engine)
- `@11ty/client/njk` (Nunjucks Template Engine)
- `@11ty/client/liquid` (Liquid Template Engine)
- `@11ty/client/i18n` (i18n Plugin)
